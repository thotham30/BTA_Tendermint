# Tendermint Visualizer (Starter)

## Quick start
1. `npm install`
2. `npm run dev`
3. Open `http://localhost:5173`

## What this includes
- Simple Tendermint-like simulator (`src/utils/tendermintSimulation.js`)
- React UI visualizing validators, messages, commits
- Controls to play/pause/step, adjust validator count, faulty count, latency
